# Public Services Intelligence Map

Projeto de visualização geoespacial e análise inteligente de serviços públicos utilizando dados oficiais e APIs abertas.

## Tecnologias
- Python
- Streamlit
- Deck.GL + Mapbox
- IBGE API
- Brasil.io

## Objetivo

Apoiar a gestão pública por meio de dados, identificando regiões com maior demanda e déficit de serviços.

## Aplicação

Planejamento urbano, políticas públicas e alocação eficiente de recursos.
